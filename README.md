# Competitive Programming in Rust

## About
Rust > C++.

## How to use
0) Learn cp
1) Clone this repo.
2) Install `cargo` and `cargo-generate`
3) `bash new-contest.sh`
4) Implement `a.rs`
5) `cargo r < input.txt`
6) `bash build.sh`
7) Submit `solution.rs`
8) Update `main.rs` to say `mod b` and `b::solve()`
9) Goto 3
